# 高校教材管理系统简介 java980

## 项目概述

高校教材管理系统是一个基于Node.js的全功能管理系统，采用前后端分离的架构设计。系统支持管理员、教师、学生三种角色，提供全面的教材管理、出入库记录、领书分析等功能。

## 技术栈

- 前端：Vue
- 后端：Egg.js
- 数据库：MySQL
- Node版本：12

## 功能模块

### 系统角色

- 管理员
- 教师
- 学生

### 管理员功能

- 用户管理
- 角色管理
- 菜单管理
- 教材数据
- 教材征订
- 出入库记录
- 领书操作
- 领书分析
- 班级管理
- 课程信息
- 教师信息

### 教师功能

- 首页
- 教材数据
- 教材征订
- 出入库记录
- 领书操作
- 领书分析
- 班级管理
- 课程信息

### 学生功能

- 教材数据
- 出入库记录
- 领书分析
- 教师信息

## 系统特点

- **前后端分离架构**：前端Vue与后端Egg.js分离设计，便于维护和扩展。
- **完整项目源码+数据库SQL**：提供完整的开发资源和数据库结构。
- **多角色支持**：系统支持管理员、教师、学生等多种角色。
- **功能齐全**：涵盖教材管理、出入库管理、领书分析等全面功能。

## 运行环境

- Node.js 12.x
- MySQL

## 目录结构

```plaintext
.
├── README.md

└── database           // 数据库相关文件
```

## 部署步骤

1. 环境准备：安装Node.js 12.x版本和MySQL数据库。
2. 数据库搭建：导入`database`目录下的SQL文件到MySQL数据库。
3. 启动后端服务：进入`server`目录，执行`npm install`安装依赖，然后执行`npm run start`启动服务。
4. 启动前端服务：进入`client`目录，执行`npm install`安装依赖，然后执行`npm run serve`启动服务。

## 核心亮点

- **灵活的角色权限管理**：对不同角色赋予不同权限，实现精准管理。
- **全面的功能覆盖**：满足教材管理的各项需求，提高工作效率。
- **分离的开发模式**：前后端分离，提升开发效率，降低维护成本。
